Анализатор ZB

- Краткое описание
* В папку input положить файлы логов;
* В папке output после выполнения программы появятся плоды;
* Внешний вид плодов частично задаётся шаблоном в корне 160123.xlsx,
  Подробнее о шаблоне ниже;

- Запускать командой
> python zbAnalyser.0.0.12.py
Или выполнить zbAnalyser.bat

- Установка
Перед запуском убедиться, что установлена среда исполнения Python и 
установлены пакеты:
* et-xmlfile (1.0.1)
* jdcal (1.2)
* openpyxl (2.3.2)

Для установки среды исполнения
* скачать и установить последний дистрибутив Python 3.x с
  официального сайта (https://www.python.org/downloads/)
* запустить install.bat
  (требуется доступ к репозиторию сообщества)
* или установить недостающие пакеты вручную с помощью команды
  > pip install <Имя_Пакета>
  (требуется доступ к репозиторию сообщества)
* если доступа к репозиторию сообщества не наблюдается, то
  скачать соотвествующие архивы с официального сайта
  (https://pypi.python.org/pypi?%3Aaction=browse),
  распаковать, найти внутри и выполнить setup.py
  Замечание: в пакете openpyxl может не быть setup.py,
  в этом случае, просто положить папку openpyxl в корень

- Подробнее
* <...в работе...>.
  В выводе команды alt производится поиск Alarm'ов из файла 
  Alarms_and_events.xlsx.
* Файл шалона состоит из трёх листов:
  1) Front Sheet - на нём есть placeholder'ы, суть которых  довольно
     очевидна, есть комментарий, помечающий строку для копирования.
     Скопированы будут только ячейки со значениями.
     Важно: имена функций ячеек Excel следует писать на
     английском языке.
  2) Error list. Summary - пока просто лист. Сюда в будущем будут
     выведены подробности найденных ошибок.
  3) Controller log template - шаблонная страница отчёта по
     контроллеру. Будет удалена в результирующем файле.
     На ней тоже имеются placeholder'ы.
     Скопирована по мере необходимости будет строка №5.
* На данный момент адекватно отрабатывает только пункт "Check active Alarms" и "Health check scheduler".

- Замеченные баги и ошибки
* После работы программы Excel открывается с ошибкой, что обнаружено
  содержимое, которое не удалось прочитать. Нажать "Да".
  В результате откроется файл без комментариев.
